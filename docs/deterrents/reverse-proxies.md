# Reverse proxies

- L3/L4 at least. L7 preferred.
- AKA: Load balancers, CDNs.

## Features

- Unmetered inbound traffic.
- DDoS protection.
- Firewalls, including Web Application Firewalls (WAFs). Some support managed rulesets.
- Caching.
- TSL termination.
